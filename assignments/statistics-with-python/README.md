# 📘 Assignment: Statistics with Python

## 🎯 Objective

Master statistical analysis using pandas and numpy to explore, analyze, and interpret data sets with practical statistical methods.

## 📝 Tasks

### 🛠️ Perform Statistical Analysis on Real Data

#### Description
Build a Python application that loads a dataset and performs comprehensive statistical analysis including descriptive statistics, hypothesis testing, correlation analysis, and data exploration. The analysis should generate reports and visualizations summarizing key findings.

#### Requirements
Completed program should:

- Load and explore data from CSV files.
- Calculate descriptive statistics: mean, median, mode, standard deviation, variance, quartiles.
- Identify and handle outliers using IQR or Z-score methods.
- Perform correlation analysis and create correlation matrices.
- Conduct hypothesis tests (t-test, chi-square, etc.) where appropriate.
- Group and aggregate data using groupby operations.
- Calculate confidence intervals for key metrics.
- Identify trends and patterns in the data.
- Generate a comprehensive statistical report (text or HTML).
- Create visualizations of distributions, correlations, and trends.
- Handle missing values appropriately (drop, fill, interpolate).
- Compare statistics across different subgroups.

Example usage:

```bash
python starter-code.py student_grades.csv
# Output: Creates analysis_report.txt and statistical_plots.html
```

Analysis includes:
- Summary statistics per column
- Outlier detection
- Correlation analysis
- Distribution analysis
- Comparison of groups

Starter code: see `starter-code.py` in this folder.
